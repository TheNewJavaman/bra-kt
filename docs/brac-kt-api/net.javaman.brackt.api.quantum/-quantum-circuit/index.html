A named set of registers that hold Qubit and Bit lists

To randomize three qubits:

val qc = QuantumCircuit(name = "Basic Circuit", numQubits = 3, numBits = 3)
qc.compose {
    h(0)
    h(1)
    h(2)
}

Or use Kotlin's built-in functions:

val qc = QuantumCircuit(name = "Basic Circuit", numQubits = 3, numBits = 3)
qc.compose {
    repeat(3) { h(it) }
}
